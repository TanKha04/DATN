NHIỆM VỤ CÁC CÔNG NGHỆ TRONG TRANG WEB
========================================

1. Bootstrap
   - Thiết kế giao diện responsive, đẹp mắt
   - Tương thích đa thiết bị (điện thoại, máy tính)

2. REST API
   - Chuẩn giao tiếp giữa giao diện và server
   - Xử lý các thao tác như đăng bài, nhắn tin, ứng tuyển

3. PHP
   - Ngôn ngữ lập trình phía server
   - Xử lý logic nghiệp vụ (đăng nhập, đăng ký, quản lý bài đăng...)

4. MySQL
   - Cơ sở dữ liệu lưu trữ thông tin
   - Lưu trữ: người dùng, bài đăng, tin nhắn, đơn ứng tuyển

5. Postman
   - Công cụ kiểm thử API
   - Đảm bảo các chức năng hoạt động đúng trước khi triển khai

6. Docker
   - Đóng gói ứng dụng thành container
   - Giúp triển khai dễ dàng trên mọi môi trường
